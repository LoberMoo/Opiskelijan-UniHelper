# Testaussuunnitelma<br><br>


Sovelluksen testaus on toteudettu käyttämällä Robot Framework skripti-kieltä.<br>

Sovelluksen testausta varten yritämme luoda mahdollisimman monelle tärkeälle käyttötapaukselle testaus automaatti-ohjelman.<br><br>

Tämän hetkiset käyttötapaukset joita on testattu ja joista on testejä ovat:<br>
- Kirjautuminen
- Olotilan merkkaaminen
- Päiväkirjamerkin lisäys.
<br>
<br>
Koitamme tulevaisuudessa tehdä lisää automaatio testejä sovellusta varten.
