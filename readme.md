## Toiminnallisuudet

- **Kirjautuminen** Kubios tunnuksilla kirjautuminen
- **Etusivu** Palautumisprosentti ja HRV-data Kubios APIsta, olotila-arviointi
- **PSQI-kysely** Pittsburgh Sleep Quality Index -unenlaatuindeksi pisteytyksineen
- **BMI-laskuri** BMI laskeminen värikoodattuna tuloksena
- **Päiväkirja** Päiväkirjamerkintöjen lisäys ja tarkastelu
- **Palaute** Palautelomake tiedostoliitteellä
- **Sivupalkki** Animoitu hampurilaisvalikko navigointiin
