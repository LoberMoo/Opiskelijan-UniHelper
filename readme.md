# Opiskelijan-UniHelper

Metropolia AMK:n Hyvinvointi- ja terveysteknologian RYHMÄ 3 ryhmätyö kevät 2026. Sovellus seuraa käyttäjän dataa ja hyvinvointia Kubios HRV -datan avulla.

- **Sovelluksen linkki**: [https://kivi-server.swedencentral.cloudapp.azure.com/]

- **Back-end API:** [https://github.com/LoberMoo/Opiskelijan-UniHelper/tree/main/Back-end]
- **Sovelluksen dokumentaatiot: (Apidoc ja testaukset):** [https://lobermoo.github.io/Opiskelijan-UniHelper/]

## Toiminnallisuudet

- **Kirjautuminen** Kubios tunnuksilla kirjautuminen
- **Etusivu** Palautumisprosentti ja HRV-data Kubios APIsta, olotila-arviointi
- **PSQI-kysely** Pittsburgh Sleep Quality Index -unenlaatuindeksi pisteytyksineen
- **BMI-laskuri** BMI laskeminen värikoodattuna tuloksena
- **Päiväkirja** Päiväkirjamerkintöjen lisäys ja tarkastelu
- **Palaute** Palautelomake tiedostoliitteellä
- **Sivupalkki** Animoitu hampurilaisvalikko navigointiin
